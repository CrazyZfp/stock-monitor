Stock Monitor

—
代码 名称 昵称 报价 当前涨速 时间 启用 操作 做T

暂无监控股票，点击"新增股票"开始

支持以下占位符，不存在的占位符会导致保存失败。可写多行，每行一个模板随机选用。

查看占位符说明
占位符说明可用类型备注
{name}官方股票名称全部不含昵称
{nickname}昵称全部未设置时回退到 {name}
{price}当前价格全部格式如 19.80
{threshold}触发阈值price_high / price_low单档绝对价格；其他类型为空
{daily_change}当日涨跌幅全部基于昨收的当日涨跌百分比，格式如 +2.50%
{speed_change}涨速surge_up / surge_down涨速窗口内的涨跌百分比，格式如 +2.50%
{retracement}回撤百分比retracement从最近峰值回落的百分比，格式如 -3.00%
{bounce}反弹百分比bounce从最近谷值回升的百分比，格式如 +3.00%
{time}涨速窗口（分钟）全部取自股票配置的"窗口/分"字段
{tier_index}档位序号 (1, 2, 3…)daily_up / daily_down当日涨跌百分比多档中的第几档
{tier_threshold}档位阈值daily_up / daily_down该档位的涨跌百分比，格式如 5.00%
{peak_price}回撤前峰值retracement最近一次突破后达到的最高价
{valley_price}反弹前谷值bounce最近一次跌破后达到的最低价
{t_type}做T操作类型t_sell / t_buyS 表示先卖后买，B 表示先买后卖
{t_price}做T事件价格t_sell / t_buy添加 S/B 事件时记录的价格
{t_threshold}做T百分比t_sell / t_buy取自股票配置的做T阈值，格式如 3.00%

可从钉钉群机器人设置中获取。空字符串表示不发通知。
